## Features

- Subdomain enumeration (`crt.sh`)
- Bruteforce common paths
- robots.txt & sitemap.xml fetcher
- Header-based technology detection
- IP resolution
- CLI with arguments for modular use
- Colorful terminal output
- Installable via `pip`
